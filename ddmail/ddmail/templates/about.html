About

Here you can find information regarding our e-mail service.

What is ddmail.se

A reasonably nice e-mail service. Who value Privacy and Security more than speed, design and avalability.

Privacy

We try to honner your privacy. We do not track you in any way. We do not log or save your ip in any way. We do not serve ads. We do not use javascript. The service is avalible as a tor onion service(before known as tor hidden service). The service is avalible as a i2p service. No personal information is needed for registration. We provide anonymous payment options.

Security

We try to follow the principle of less code, less attacksurface. The service is designed with the principel of leased privileged. The server configuration and code is all open source and can be reviewed and used by anyone. We use automatic testing. We see all input as evil. We use hard to crack password hashing with argon2id. We autoupdate the server software every hour. If there was a kernel update we reboot our servers once a day. The service is run on Debian 10. We use mod_security for WAF. We use apparmore as MAC. We do not provide any webmail interface.

Features

Unlimited number of e-mails accounts per account. Unlimited number of aliases per account. Unlimited number of own domains per account. Avalible as Tor Onion Service. Avalible as i2p service. Protocol supported for getting emails is imaps.
